# SchoolGenius

## Project info

A modern School Management System built with React, Vite, and Tailwind CSS.

## Getting Started

Prerequisites: Node.js & npm installed.

```sh
# Step 1: Install dependencies
npm i

# Step 2: Start the development server
npm run dev

# Step 3: Build for production
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
